# Freyr

一个关于图书的小应用，主要功能有收藏图书，发布书评，点赞评论等。前端用到了React与Mobx，后端使用Express框架，数据存储使用leanCloud。

## Usage

由于未上传leanCloud私人账号信息，所以本地无法开启项目，后期会增加新功能并部署到服务器上。
